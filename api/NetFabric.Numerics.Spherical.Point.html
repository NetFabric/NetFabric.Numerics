Reduce<TAngleUnits, T>(Point<TAngleUnits, T>)

Reduces a point in spherical coordinates by applying reduction functions to its components.

public static PointReduced<TAngleUnits, T> Reduce<TAngleUnits, T>(Point<TAngleUnits, T> point) where TAngleUnits : IAngleUnits where T : struct, IFloatingPoint<T>, IMinMaxValue<T>

Parameters

point Point<TAngleUnits, T>

The input point in spherical coordinates to be reduced.

Returns

PointReduced<TAngleUnits, T>

A new PointReduced<TAngleUnits, T> object with a reduced azimuthal angle and polar angle, while keeping the original radius.

Type Parameters

TAngleUnits

The type representing angle units.

T

The type used for the coordinates.

Remarks

The reduction process involves applying reduction functions to the azimuthal and polar angles of the input spherical point. The radius component remains unchanged.

ToRectangular<T>(Point<Radians, T>)

Converts a point in spherical coordinates to rectangular 3D coordinates.

public static Point<T> ToRectangular<T>(Point<Radians, T> point) where T : struct, IFloatingPoint<T>, IMinMaxValue<T>, ITrigonometricFunctions<T>

Parameters

point Point<Radians, T>

The point in spherical coordinates to convert.

Returns

Point<T>

The rectangular 3D coordinates representing the point.

Type Parameters

T

The type used for the coordinates.

Remarks

If the type of point to convert doesn't meet the constraints, please convert it first to a suitable type using one of the conversion methods:

  • CreateChecked<TOther>(in Point<TAngleUnits, TOther>)
  • CreateSaturating<TOther>(in Point<TAngleUnits, TOther>)
  • CreateTruncating<TOther>(in Point<TAngleUnits, TOther>)
