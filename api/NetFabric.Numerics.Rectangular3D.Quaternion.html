Remarks

The dot product of two quaternions represents the cosine of the angle between them. It is calculated by multiplying the corresponding components of the quaternions and summing the results.

dot = (left.X * right.X) + (left.Y * right.Y) + (left.Z * right.Z) + (left.W * right.W)

The dot product can be useful for various quaternion operations, such as determining if two quaternions are parallel or perpendicular to each other.

Type Parameters

Remarks

The inverse of a quaternion represents the opposite rotation. When a quaternion is multiplied by its inverse, the result is the identity quaternion (1, 0, 0, 0). The inverse of a non-zero quaternion is obtained by negating the vector part (x, y, z) and dividing each component by the squared magnitude.

Q^-1 = (x, y, z, w)^-1 = (-x / |Q|^2, -y / |Q|^2, -z / |Q|^2, w / |Q|^2)

Note that the inverse of a quaternion is only defined when the quaternion is non-zero. In the case of a zero quaternion, attempting to calculate the inverse will result in division by zero, which is an undefined operation.
