Remarks

In a 3D Rectangular coordinate system, a point is represented by a triplet of values (X, Y, Z) that specify its position in 3D space. The X-coordinate represents the horizontal position, the Y-coordinate represents the vertical position, and the Z-coordinate represents the depth or altitude.
