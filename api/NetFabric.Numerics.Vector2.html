Remarks

This method adds the corresponding coordinates of the two input vectors and returns a new Vector2 with the resulting sums. A new vector that is clamped within the specified minimum and maximum values for each component. If any component of the vector is less than the corresponding component in min, the minimum value is used. If any component of the vector is greater than the corresponding component in max, the maximum value is used. Otherwise, the original vector is returned.

Type Parameters

T

The numeric type used internally by vector.

Remarks

The Clamp<T>(in Vector2<T>, in Vector2<T>, in Vector2<T>) method ensures that each component of the resulting vector is within the specified range. If any component of the vector is less than the corresponding component in min, that component is clamped to the minimum value. If any component of the vector is greater than the corresponding component in max, that component is clamped to the maximum value. Otherwise, if all components of the vector are already within the range, the original vector is returned.

This method is useful when you want to restrict a 3D vector to a certain range for each component.

Lerp<T>(in Vector2<T>, in Vector2<T>, T)

Performs linear interpolation between two vectors.

public static Vector2<T> Lerp<T>(in Vector2<T> start, in Vector2<T> end, T factor) where T : struct, IFloatingPoint<T>, IMinMaxValue<T>

Parameters

start Vector2<T>

The starting vector.

end Vector2<T>

The ending vector.

factor T

The interpolation factor. Should be between 0 and 1.

Returns

Vector2<T>

A new vector that represents the interpolated value.

Type Parameters

T

The type of the vector components.

Remarks

This method performs linear interpolation between the start and end vectors using the specified factor. The factor should be a value between 0 and 1, where 0 represents the start vector and 1 represents the end vector. The resulting vector is calculated by multiplying the start vector by (1 - factor), multiplying the end vector by factor, and then adding the two resulting vectors together.

Normalize<T>(in Vector2<T>)

public static Vector2<T> Normalize<T>(in Vector2<T> vector) where T : struct, INumber<T>, IMinMaxValue<T>, IRootFunctions<T>

Parameters

vector Vector2<T>

The vector to normalize.

Returns

Vector2<T>

A new vector that represents the normalized form of the specified vector.

Type Parameters

T

Remarks

The Normalize<T>(in Vector2<T>) method calculates a normalized form of the specified vector. The resulting vector will have the same direction as the original vector but will have a magnitude of 1.

To normalize a vector means to scale it to a magnitude of 1 while preserving its direction. This is useful when you want to work with direction vectors or ensure consistent scaling across different vectors.

Note that if the specified vector is a zero vector (all components are zero), the method will return the zero vector itself, as it cannot be normalized.
