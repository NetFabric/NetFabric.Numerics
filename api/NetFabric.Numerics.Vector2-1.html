Parameters

right Vector2<T>

The right Vector2<T> to compare.

Returns

bool

True if the magnitude of the left Vector2<T> is greater than the magnitude of the right Vector2<T>; otherwise, false.

Remarks

This operator compares the magnitude of the left Vector2<T> with the magnitude of the right Vector2<T>. It returns true if and only if the magnitude of the left Vector2<T> is greater than the magnitude of the right Vector2<T>. Otherwise, it returns false. The comparison is based on the magnitudes of the vectors and is independent of their directions.

The comparison of magnitudes is performed using the Magnitude<T>(in Vector2<T>) method, which calculates the square root of the magnitude squared. However, to optimize the performance, this operator directly compares the magnitude squared values of the vectors, accessible through the MagnitudeSquared<T>(in Vector2<T>) method.
